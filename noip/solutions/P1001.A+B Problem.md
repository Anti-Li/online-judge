### P1001 A+B Problem

算法标签: **模拟**

#### Description

基本输入输出题目

#### Analysis

回顾一下 C/C++ 的输入输出

```c
#include <stdio.h>

int main() {
    int a,b;
    scanf("%d%d",&a,&b);
    printf("%d", a+b);
    return 0;
}
```

```cpp
#include <iostream>
#include <cstdio>

using namespace std;

int main() {
    int a,b;
    cin >> a >> b;
    cout << a+b;
    return 0;
}
```
